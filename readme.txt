An esential helper tool for woocommerce single product page. Borderless freedom to customize single product page.

== Description ==

Single product page customizer plugin is a nice tool for WooCommerce Single product page. User can easily customize there Single product page. They can show/hide meta section, enable/disable plus-minus button, hide price for guest user and many more.

== Installation ==

1. **Upload the Plugin Files**:
   - Download the plugin ZIP file.
   - Go to your WordPress admin dashboard.
   - Navigate to `Plugins > Add New > Upload Plugin`.
   - Upload the ZIP file and click "Install Now."
2. **Activate the Plugin**:
   - After installation, click "Activate Plugin."
3. **Install Required Plugins**:
   - Ensure WooCommerce is installed and activated.
4. **Configure the Plugin**:
   - Go to `Dashboard > WooCommerce > Products` to customize the plugin.
